## Frameworks and Technologies Used:
* GraphQL
* ExpressJs
* Reactjs
* React-Apollo
* Node
* NPM
* HTML, CSS

## To run locally
* Download/Pull the master branch
* Redirect terminal to the project folder
* Redirect to BackEnd
* Type 'nodemon server' and hit enter
* Go to client and hit 'npm start'

Make sure Mongo server is running.
